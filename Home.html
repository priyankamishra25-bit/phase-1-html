About This Website

The student profile and course portal is a simple multi-page website built using only html. It helps students manage and view their personal information,enrolled courses,grades,and feedback all in one place.Each page is connected with easy navigation links so that users can move between sections smoothly.This project focuses on understanding the basic structure of web pages and how different HTML elements work together to create a complete organized website.
